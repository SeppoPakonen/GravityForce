Willkommen zu Gravity Strike!
Welcome to Gravity Strike!


Das /doc-Verzeichnis enthaelt umfangreiche Dokumentationen, teilweise
jedoch nur in Deutsch.

The /doc directory contains much documentation, but some of them is 
only available in German language yet.


  /doc/ger/
  
  gsdocger.txt          -> Haupt-Readme-Datei
  history.txt           -> Versions-History
  bugs.txt              -> bekannte Bugs
  faq.txt               -> "Wissenswertes"
  shipimg.txt           -> Erstellung eines eigenen Schiffes
  maps.txt              -> Erstellung eigener Karten
  script.txt            -> Erstellung eigener Levelscripte
  funclib.txt           -> Erklaerung der zus. Scripting-Funktionen
  scriptex.txt          -> Beispiele fuer Levelscripte
  tips.txt              -> Tips fuer Levelscripter

  /doc/eng/
  
  gsdoceng.txt          -> Main readme file
  history.txt           -> Version history
  faq.txt               -> Frequently Asked Questions
  

Und wer einfach nur spielen moechte, startet "gs.exe" :-)
And if you just wanna play, start "gs.exe" :-)
